Privacy policy for the mobile application BikeSharingHub

This privacy policy governs your use of the software application BikeSharingHub (the "Application") for mobile devices that is maintained by François FERREIRA DE SOUSA and contributors ("Us").

The Application displays data about "bikesharing networks". A bikesharing network means a set of stations located in a specific city or area, where self-service bicycles can be rented through a subscription to a given operator. We are not affiliated to any of these operators, all data is obtained freely.

1. What information does the Application obtain and how is it used?

1.1. User location

During the installation or usage of the Application, you must decide to grant or deny access to your location. When granted, the Application offers the following features :

  • Filter the display of nearby stations
  • Display your location on a map
To perform these features, your location is processed only on your device. It is not collected by Us, neither transmited to any third party.

1.2. List of favorites stations

During the usage of the Application you may mark some stations as favorites. A given station might be associated to its location.

This list of stations and their location is stored only on your device memory. It is not collected by Us neither transmited to any thirdparty.

1.3. List of bikesharing networks

During the first launch of the Application, you must select the list of bikesharing networks that you want to monitor. From then on, the Application sends regularly a request with the content of this list to a third-party ("The API server") in order to refresh data related to them.

The API server could be either one you provided yourself, or the default one which is a free to use server hosted at https://citybik.es (no privacy policy found). Please note that We provide only the Application and We are not affiliated to The API server you will use. We don't send intentionaly any personal data to it but you are informed that, technically The API server can see your IP address.

1.4. Browsed map areas

During the usage of the Application you may use a feature which displays on a map the station's location, and eventually your own location. The map imagery is not embedded in the Application. As you browse the map, the corresponding image is downloaded from a third-party server ("The tile provider").

The tile provider is one of the following (depending on your Application's settings) :

  • "Mapnik" operated by "OpenStreetMap fondation" (privacy policy available at https://wiki.osmfoundation.org/wiki/Privacy_Policy)
  • "ÖPNKarte" operated by "Memomaps" (privacy policy available at : https://www.xn--pnvkarte-m4a.de)
  • "CyclOSM" operated by "OpenStreetMap France" (no privacy policy found)
Please note that We provide only the Application and We are not affiliated to any tile provider. We don't send intentionaly any personal data to them, but you are informed that, technically the Tile provider can see your IP address and the location area you have downloaded.

2. Changes

This Privacy Policy may be updated from time to time for any reason. You can review all changes ever made on the current page by visiting https://github.com/francoisfds/francoisfds.github.io/commits/master/docs/bikesharinghub/privacy-policy.html
